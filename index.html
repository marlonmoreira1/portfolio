Sistema de Recomendação

O objetivo desse projeto é criar um sistema que possa recomendar produtos que são comprados em conjunto com muita frequência. Para isso vou usar o dataset Online-Retail que está disponível na UCI Machine Learning Repository. http://archive.ics.uci.edu/ml/datasets/online+retail. Esse conjunto dados contém todas as transações ocorridas entre 01/12/2010 e 09/12/2011 de uma empresa de varejo online do Reino Unido. Para esse projeto foi usado o pacote arules da linguagem R. Aqui você encontra o script com os códigos em R e um relatório em pdf explicando todo o processo.

Estratégia para um Portfólio de ações

Utilizando dados históricos (01-06-2013 até 30-06-2017) de uma bolsa de ações, gerei sinais de trading baseados em um indicador de momento (valor ajustado do preço da ação ao final do dia). Após gerar sinais, calculei os retornos e testei se de fato essa estratégia de momentum poderia ser rentável. Por fim, utilizando o algoritmo Prophet da biblioteca do Facebook Prophet, fiz previsões para os próximos 6 meses e avaliei a tendência de algumas ações.
